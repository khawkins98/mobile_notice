
Module: Mobile Notice mobile_notice
Author: Ken Hawkins <http://AllAboutKen.com>


Description
===========
A simple JS detect that shows a notice to use the mobile app on a mobile-sized screen.

Requirements
============

* JS.


Installation
============
* Copy the 'mobile_notice' module directory in to your Drupal
sites/all/modules directory as usual.


Usage
=====
Effect to page loads is automatic, just enable.
